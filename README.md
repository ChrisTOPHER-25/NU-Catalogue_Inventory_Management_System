# NUCATALOGUE: WEB-BASED INVENTORY MANAGEMENT SYSTEM FOR NU LAGUNA MERCHANDISE

![PHP Logo](https://upload.wikimedia.org/wikipedia/commons/2/27/PHP-logo.svg)

## Requirements

- XAMPP Version: Recent

## Setup

- Clone project
- Copy and paste Source Code to the htdocs
- Open the Browser
- Write localhost/phpmyadmin
- Insert the Database
- Write localhost/NUCatalogue to Run the System
